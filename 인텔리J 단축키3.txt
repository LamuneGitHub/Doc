## 선택 
	Ctrl + w	:	선택확장 ( 단어 ->... 계속 범위 확장 )
	Shift + end	:	선택된 현재 커서 까지 end 곳 
	Shift + home	:	선택된 현재 커서 까지 home 곳 
	Ctrl + [ or ]	:	이동 커서 까지 장 초 / 말 괄호 곳

	Ctrl + BackSpace	:	커서 앞 단어 삭제 

	Ctrl + Alt + V	:	 클립보드?

	Ctrl + W	:	Surround with

	Ctrl + space	: 자동완성
	Ctrl + Shift + space	:	스마트 자동완성
	Ctrl + Shift + Enter	:	Statemet 자동완성
	Alt + Enter 	:	코드  변경 제안~  ㅇ

##  검색/이동 기능
	Ctrl + N : 클래스명으로 검색
	Ctrl + E : 최근 이용한 파일 목록
	Ctrl + Shift + A : action 검색
	Ctrl + Shift + F :  파일에서 찾기
	Ctrl + F12  : 파일 Out Line 검색 팝업
	Shift 더블클릭	:  모든것에 대한 검색기능
	
		 * 검색 옵션
		  /  := 폴더
		  검색조건: 숫자   :=  클래스  + 라인 이동

	Alt + F3	:	선택한 텍스트, 하나하나 아래로 찾기 같은 텍스트 및 강조
	Ctrl + Shift + F7	:	동일한 단어 강조 ( ESC로 해제 ) 

	Ctrl + Alt + 좌/우 : 이전 위치
	Ctrl + Alt + 위/아래 : Find 에서 이전/다음항목
	Alt + F7 :  변수 / 클래스 호출된 곳 
	Ctrl + Alt + F7 :  변수 / 클래스 호출된 곳 	
	Ctrl + U	:	선택 부모 클래스로

	Ctrl + Q 마우스 변수 / 类名 / 방법 명 등 위에 놓다 (해도 지금 팁 보충 때 따라) 문서 내용 보이기 같은 비슷한 기능 그리고 하나Ctrl + Shift + I 

	Ctrl + B 커서 아래 클래스 혹은 방법은 바로 열기 (대하여 전면 지원 잘 수 있는 name, 예를 들면 직접 통해 class 위치 까지 css 파일 위치) (에 ctrl+ 커서 지향)
